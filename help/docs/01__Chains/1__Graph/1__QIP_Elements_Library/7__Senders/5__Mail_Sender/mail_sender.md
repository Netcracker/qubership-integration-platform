# Mail Sender
## Description

---
**Mail Sender** element provides an ability to pass the e-letter over configured mail server.

## User Interface

---
### "Parameters" Tab

| Parameter | Mandatory | Data Type | Description                                                                                                                                                                                                                                                                               | Sample                     |
| --------- | :-------- | :-------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | -------------------------- |
| URL       | M         | String    | Target mail server URI.<br>**Format:**<br><pre style="background-color: #F5F5F7"><code  style="color: #000000">{protocol}://{host}:{port}</code></pre>Supported protocols are SMTP and SMTPS (secure variant of SMTP layered over SSL). Default ports are 25 for SMTP, and 465 for SMTPS. | smpts://mailserver.com:465 |

Corresponds to "Subject" section of the e-letter.                                                                                                                                                                                                                      | Some Subject               |
